**Problems**

or not .

**Solution**
* learnt about the specific keywords which was used to write the code,they are as follows :
* `readdir`-for reading the directory
* `\check`-for accesing the request
*  `"rev-parse --is-inside-work-tree"`-for checkingif its a git repo
* `stats && isDirectory`-for checking if its a  valid directory.
